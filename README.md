# 植物战队 — 日夜防守网页模拟器

一款纯静态网页版塔防小游戏（HTML / CSS / JS），支持日夜循环、植物收集与种植、怪物刷新与对战，可直接部署到 GitHub Pages。

## 本地运行

直接双击 `index.html` 即可打开，或在 VS Code 里用 Live Server。

## 发布到 GitHub Pages

1. 新建 GitHub 仓库（例如 `PlantBattle`）。
2. 上传本目录全部文件：
   - `index.html`
   - `style.css`
   - `app.js`
3. 进入仓库 `Settings -> Pages`。
4. `Build and deployment` 选择：
   - Source: `Deploy from a branch`
   - Branch: `main` / `root`
5. 保存后等待 1-2 分钟，访问生成的页面链接。

## 游戏流程

1. **白天阶段**（30 秒）：收集区会刷新随机植物，点击即可收入背包；从背包选中植物后点击种植区空格放置。
2. **夜晚阶段**：怪物从刷怪区向种植区推进，种植好的植物自动攻击来袭怪物。
3. 每回合怪物数量与属性随回合数递增，挑战难度逐步提升。
4. 怪物突破种植区会扣除生命值，生命值归零则游戏结束。

## 配置系统

### 植物库配置
在页面下方「植物库配置」面板中可以：
- 查看 / 编辑 / 删除已有植物
- 添加自定义植物
- 为每个植物设置：名字、图片 URL、定位（防御 / 输出 / 辅助）、攻击方式（近战 / 远程 / 范围）、HP、ATK、DEF、暴击率、暴伤倍率、闪避率、技能（名称 / 类型 / 系数 / 冷却）

### 怪物库配置
在「怪物库配置」面板中可以：
- 查看 / 编辑 / 删除已有怪物
- 添加自定义怪物
- 为每个怪物设置：名字、Emoji、HP、ATK、移动速度、攻击间隔、击杀奖励

### 保存 / 加载
点击顶栏「保存配置」将植物库与怪物库存入浏览器本地存储；点击「加载配置」可恢复之前保存的配置。

## 战斗规则

- 种植区有 5 列 × 2 行共 10 个槽位。
- 伤害流程：闪避判定 → 暴击判定 → 防御减伤（最低 1 点伤害）。
- 技能类型：
  - **普通伤害**：按技能系数倍增输出。
  - **中毒 DoT**：附加持续中毒伤害。
  - **护盾**：为自身附加吸收护盾。
  - **减速**：降低怪物移动和攻击速度。
- 防御型植物在前排可为后排吸收伤害。
